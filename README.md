## ❗️Usage (without need installation):
```bash
python3 src/moo.py
```

## ❗️ Install:

### Linux
```bash
make install
```

### Windows
1. `pip3 install -U pyinstaller`

2. `pyinstaller --onefile src/moo.py`

3. Move to executables folder and use from console.
